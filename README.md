PARA ABRIR ARQUIVOS XML (Contém Diagramas)

Acesse o site:
https://app.diagrams.net/

Vai aparecer a pergunta "Where would you like to save your diagrams?"

Escolha:

Device → se o arquivo está no seu computador
Clique em "Open Existing Diagram"

Selecione o arquivo:
.drawio
.xml
Clique em Abrir.
