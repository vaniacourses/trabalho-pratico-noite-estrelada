## 💡 Dicas Rápidas

**Problema: Porta 3000 já em uso**
```bash
npm run dev -- -p 3001
```

**Problema: PostgreSQL não conecta**
```bash
docker-compose up -d
```

**Problema: Dependências não instaladas**
```bash
npm install
```

**Visualizar dados do banco:**
```bash
npx prisma studio
```
